In collaboration with Who Targets Me, we monitored 91 political advertisers in Montenegro during the 2023 Montenegrin legislative election to better understand how campaigns use different targeting methods made available by Meta. To do this, we used data from the Meta Ad Library, using the new ‚ÄòAudience‚Äô data which gives some detail on how pages target their ads.

To better understand the election, we kept only advertisers who:

  1. Advertised in the last 7 days (May 31st - Jun 6th 2023)
  2. Advertised in the last 30 days (May 8th - Jun 6th 2023)

Note: Meta only provides 7, 30 and 90 days windows for the targeting data in their Ad Library. Meta‚Äôs data also lags by a few days. Report will be updated as soon as new data is available.
